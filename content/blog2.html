Name 5 commands you used, and what they do.

  • 1.`cd` change directory: Allows you to move into the folders of your computer.
  • 2.`ls` list: Will list everything inside of that particular folder / directory.
  • 3.`mkdir` make directory: Will make a folder / directory inside of the directory you are currently in.
  • 4.`touch` Make file: Will make a file inside of the folder / directory you are currently inside of.
  • 5.`cat` Read document: Cat will show you / read out the content of any file in any directory.
