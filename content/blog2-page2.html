What's the difference between git and GitHub?

Git is a programme used to remember, manages and control the command line and different sets of code. Github is an open source website which allows uses to share, track and work simultaneously on code projects without conflict.

Can you think of an analogy to describe them?

If the computer were a human brain, git would be the part that compartmentalises and comprehends the information it is given. While GitHub is like a social media platform, where we can share(push) the thoughts(data/code) that the brain(git) has conceptualised(committed).

What is a Git workflow?

A series of steps regarding git and its commands which allow developers to maintain order when working with big blocks of code.
